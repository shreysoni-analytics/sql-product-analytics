# SQL Product Analytics

This repository contains SQL examples used to answer core product analytics questions like
growth, retention, funnel conversion, and revenue metrics.

## Topics Covered
- Joins, CTEs, window functions
- Cohort retention analysis
- Funnel analysis
- KPI definitions (DAU/WAU/MAU)

## Structure
- `schema/` includes table definitions
- `queries/` includes analytics queries
